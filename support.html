Frequently Asked Questions

How do I get started with Odoji?

1. Download Odoji from the App Store
2. Open the app and go to Settings
3. Enter your Replicate API key
4. Create your first character and start chatting!

Where can I get a Replicate API key?

Visit Replicate.com to create an account and generate an API key. The DeepSeek V3 model used by Odoji requires a valid API key.

How many characters can I create?

There's no limit to the number of characters you can create. All characters are stored locally on your device.

What languages are supported?

Odoji supports 15+ languages including English, Chinese, Japanese, Spanish, French, German, Italian, Portuguese, Russian, Korean, Arabic, Hindi, Turkish, Polish, and Dutch.

How do I create a chat room with multiple characters?

1. Go to the Chat Rooms tab
2. Tap the + button
3. Give your room a name
4. Add up to 5 characters
5. Configure room settings like response order

Is my data secure?

Yes! All your data is stored locally on your device. We don't have access to your characters, chats, or personal information. API keys are stored securely in your device's keychain.

How do I backup my data?

Currently, data backup is handled through iOS's standard backup system. We're working on adding export/import functionality in a future update.

Troubleshooting

Characters aren't responding

• Check your internet connection
• Verify your Replicate API key is valid
• Ensure you have API credits available
• Try restarting the app

App is running slowly

• Clear old chat histories you no longer need
• Reduce the message history limit in Settings
• Ensure you're running the latest version of iOS
